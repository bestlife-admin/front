Z-index

Z-index utility classes.

Example 

        
          <div class="zi-n1"></div>
          <div class="zi-0"></div>
          <div class="zi-1"></div>
          <div class="zi-2"></div>
          <div class="zi-3"></div>
          <div class="zi-999"></div>

Sass

Maps

Spacing utilities are declared via Sass map and then generated with our utilities API.

        
          $z-indexes: (
            n1: -1,
            1: 1,
            2: 2,
            3: 3,
            999: 999
          ) !default;

Utilities API

Spacing utilities are declared in our utilities API in scss/front/_utilities.scss. Learn how to use the utilities API.

        
          "z-index": (
            property: z-index,
            class: zi,
            values: $z-indexes
          )
