Migrating to v4

Front v4.0 is a major rewrite of the entire template. The most notable changes are summarized below, followed by more specific changes to relevant components. Hence, below are the breaking changes to bear in mind when moving from v3.x to v4.0 or any older version of Front.

The most notable improvements made in Front v4.0

  • Bootstrap v5 - Built on top of the latest Bootstrap v5.
  • JavaScript - jQuery is fully dropped as a dependency and all libraries and js files are fully vanilla JS now.
  • HTML - classes and code structure optimized.
  • SCSS - all files and styled updated.
  • Headers (Navbars) - completely rewritten from scratch.
